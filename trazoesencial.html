Sistema de Gestión Musical - Spotify

Paquete: Gestión de Contenido Propio Nombre del Caso de Uso: Crear Playlist No. De Orden: 1
Prioridad: Alta
Complejidad: Baja
Actor Principal: Usuario Actor Secundario: Sistema Spotify

Objetivo

Permitir al usuario crear una nueva playlist personalizada para organizar canciones según su preferencia.

Flujo Principal

  1. Usuario: Selecciona la opción "Crear nueva playlist".
  2. Sistema: Solicita un nombre para la playlist.
  3. Usuario: Ingresa el nombre de la playlist (opcionalmente, añade una descripción e imagen).
  4. Sistema: Valida el nombre (que no esté vacío o duplicado).
  5. Sistema: Crea la playlist vacía en la cuenta del usuario.
  6. Sistema: Muestra la nueva playlist disponible para ser editada.

Flujo Alternativo - Nombre vacío

  • Se genera un nombre por defecto

    • Flujo Alternativo - Cancelación

    1. Usuario: Cancela el proceso antes de guardar la playlist.
    2. Sistema: No se guarda ningún cambio.

    Reglas de Negocio

    • Un usuario puede crear playlist colaborativas donde otros usuarios pueden agregar canciones
    • Un usuario puede tener múltiples playlists.
    • Se permite que dos playlists tengan el mismo nombre si son del mismo usuario.
    • Una playlist puede estar vacía al momento de su creación.
